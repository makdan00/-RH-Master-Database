## Project Overview
This is a high-performance PostgreSQL database designed to power the Remote Hustle Database Challenge (RHDC). It replaces fragile spreadsheets with a secure, automated system that tracks participants, manages submissions, and enforces grading rules at the database level.

## ðŸ—ï¸ How it Works (The "Smart" Features)
The Skeleton (Schema): 8 normalized tables connected via strict Foreign Keys to ensure zero data orphans.

The Safety Rails (Constraints): Hardcoded rules prevent scores > 100, duplicate emails, or invalid roles.

The Secret Diary (Audit Logs): Automated PostgreSQL Triggers capture every INSERT and UPDATE into an audit trail, ensuring 100% transparency.

Production Security: Every table is protected by Row Level Security (RLS), and the leaderboard view is hardened with security_invoker to prevent unauthorized data leaks.
